# Level 4 — Prepositions: Location, Movement, Instrument

## What this level teaches

Level 4 teaches how languages encode spatial relationships and means.
Three semantic domains, each with distinct cross-linguistic structure:

| Sub-level | Domain | Key contrast |
|---|---|---|
| 4A | Static location | German Wechselpräpositionen take **dative** when static |
| 4B | Directed movement | Same German prepositions take **accusative** when directional |
| 4C | Instrument / means | JP で vs と; ZH 用 vs lexical verb for vehicles |

The central lesson of 4A + 4B together: German uses the **same nine prepositions**
(an, auf, hinter, in, neben, über, unter, vor, zwischen) for both location and direction —
the case alone signals which reading applies. No other major language in this corpus does this.

---

## Global rules — all Level 4 files

These rules override any default. Apply without exception.

**Naturalise, don't translate.**
The same meaning expressed as a native speaker would express it.
Surface form may differ across languages.

**Japanese plain form only.** No ます/です. No keigo.
Past: た form. Ongoing: ている. Future: だろう / と思う.
Japanese MUST be written in Japanese script (kanji + hiragana + katakana). Never romaji.

**German V2 word order.** Verb is always the second constituent.
Time adverbials trigger inversion: Gestern war die Katze... (not Gestern die Katze war).

**German Perfekt for past completed action** with haben (most verbs) or sein (motion/state-change verbs):
ist gelaufen, ist gesprungen, ist gefahren (motion → sein); hat gelegt, hat gestellt (placement → haben).

**Mandarin aspect markers.** 了 for completed, 在 for ongoing, 会 for future.

**Pronoun drop.** Japanese and Mandarin omit subject/object pronouns when clear from context.

**Vary subjects across groups.** Use: Tom, Kate, the child, the teacher, the dog, the cat, the woman,
the man, she, he. Do not repeat the same subject more than twice per file.

**Do not explain grammar inside generated files.** No labels, no headings, no notes.

## 4A — Static Location

### What this teaches

The subject is at rest at a location. The location phrase uses a preposition (EN/DE)
or a postposition sequence (JP) or 在 + localizer (ZH).

German: static location always takes dative with the nine Wechselpräpositionen.
Japanese: postposition の + localizer noun + に (for existence verbs いる/ある).
Mandarin: 在 + place noun + localizer noun.

### German — dative case for static location

| Preposition | Meaning | Dative example |
|---|---|---|
| auf | on (horizontal surface) | auf dem Tisch |
| unter | under | unter dem Stuhl |
| in | in / inside | in der Tasche, im Haus |
| neben | beside / next to | neben der Tür |
| hinter | behind | hinter dem Vorhang |
| vor | in front of | vor dem Haus |
| zwischen | between | zwischen den Büchern |
| über | above (without contact) | über dem Bett |
| an | at (vertical surface / edge) | an der Wand, am Fenster |

Non-Wechselpräpositionen used in 4A:
- bei + dative: near, at (person or place): beim Bahnhof, bei der Schule
- nahe + dative: near (slightly more formal)

#### German dative article forms

| Gender | Definite dative | Indefinite dative |
|---|---|---|
| masc | dem | einem |
| fem | der | einer |
| neut | dem | einem |
| pl | den + -(e)n | — |

Plural dative adds -n to the noun if it doesn't already end in -n or -s:
den Büchern, den Stühlen, den Kindern, den Hunden.

Contractions: in + dem → im; an + dem → am; in + das → ins; an + das → ans.

### Japanese — postposition system for location

Localizer nouns: 下 (shita, under/below), 上 (ue, on/above), 中 (naka, inside),
隣 (tonari, beside), 後ろ (ushiro, behind), 前 (mae, in front of),
間 (aida, between), 近く (chikaku, near), 横 (yoko, beside/side).

Construction: NOUN の LOCALIZER に + verb of existence (いる / ある)

Examples:
- 猫は机の下にいる。(animate: いる)
- 本は机の上にある。(inanimate: ある)
- 子供はベッドの中で寝ている。(activity at location: で)

#### Japanese: いる vs ある

This distinction is obligatory and must be correct in every file.

| Subject type | Verb |
|---|---|
| Animate (people, animals) | いる |
| Inanimate (objects, plants) | ある |

ある also used for events, scheduled items: 会議がある (there is a meeting).

#### Japanese: に vs で for location

| Particle | Use | Example |
|---|---|---|
| に | existence at a location (いる/ある) | 駅にいる — is at the station |
| に | destination or goal of movement | 部屋に入る — enters the room |
| で | location of an activity (doing something there) | 図書館で勉強する — studies at the library |

This distinction surfaces often in 4A: the child IS in the classroom (に + いる) vs. the child STUDIES in the classroom (で + 勉強する).

### Mandarin — location structure

Pattern: 在 + place NP + localizer word + verb / 是 + 在 + place

Localizer words (方位词):
- 上面 / 上: on top, above
- 下面 / 下: below, under
- 里面 / 里: inside
- 旁边: beside
- 后面 / 后: behind
- 前面 / 前: in front
- 中间 / 之间: between
- 上方: above (more abstract/elevated)
- 附近: nearby

Examples:
- 猫在桌子下面。(cat under the table)
- 书在包里。(book in the bag) — 里 often sufficient without 面
- 孩子在门前等着。(child waits in front of the door)

Existence sentences: 那里有一只猫。(there is a cat there) — 有 for existence, 是 for identification.

### 4A positional verbs

Files vary the predicate verb across these options:

| English | German | Japanese | Mandarin |
|---|---|---|---|
| be (static) | sein | いる (animate) / ある (inanimate) | 在 / 是 |
| sit | sitzen | 座っている | 坐着 |
| stand | stehen | 立っている | 站着 |
| lie (reclining) | liegen | 横になっている / 寝ている | 躺着 |
| hang | hängen | かかっている | 挂着 |
| wait | warten | 待っている | 等着 |
| live (habitual) | wohnen / leben | 住んでいる | 住 / 住着 |
| work (habitual) | arbeiten | 働いている | 工作 |
| sleep | schlafen | 寝ている | 睡着 |
| hide | sich verstecken | 隠れている | 躲着 |

### 4A file format

3–5 groups. Each group: exactly 4 lines (EN / DE / JP / ZH). Blank line between groups.
Vary subject (animate/inanimate), verb, and reference noun across groups.

```

猫は机の下にいる。
猫在桌子下面。

The child hides under the bed.
Das Kind versteckt sich unter dem Bett.
子供はベッドの下に隠れている。
孩子躲在床底下。
```

## 4B — Directed Movement

### What this teaches

The subject moves toward, away from, or through a location.
German Wechselpräpositionen now take **accusative** — the same nine prepositions as 4A,
but case shifts from dative to accusative to signal direction rather than static location.

This is the core lesson: dative = static, accusative = directional. No other language in this corpus
encodes this distinction via case on the same preposition.

### German — accusative case for directional movement

Same nine prepositions as 4A, now with accusative:

| Preposition | Directional meaning | Accusative example |
|---|---|---|
| auf | onto | auf den Tisch (springen) |
| unter | to under | unter das Bett (fallen) |
| in | into | ins Haus (laufen) / in die Tasche (legen) |
| neben | to beside | neben das Bett (stellen) |
| hinter | to behind | hinter das Haus (gehen) |
| vor | to in front of | vor das Haus (fahren) |
| zwischen | to between | zwischen die Stühle (schieben) |
| über | over / to above | über die Brücke (fahren) |
| an | onto (vertical) | an die Wand (hängen) |

Non-Wechselpräpositionen for goal:
- zu + dative: to (specific person or place): zum Bahnhof, zur Schule
- nach + dative: to (city, country, named place without article): nach Berlin, nach Hause
- aus + dative: out of (enclosed space): aus dem Haus, aus der Tasche
- von + dative: from (person, non-enclosed): vom Bahnhof, von der Schule
- durch + accusative: through: durch den Park, durch den Tunnel
- entlang + accusative (post-nominal): along: den Fluss entlang, die Straße entlang
- um + accusative: around: um den Tisch, um das Haus herum

#### German accusative article forms

| Gender | Definite accusative | Indefinite accusative |
|---|---|---|
| masc | den | einen |
| fem | die | eine |
| neut | das | ein |
| pl | die | — |

Note: feminine and neuter forms are identical to nominative. Only masculine changes (der → den).

Contractions: in + das → ins; an + das → ans; auf + das → aufs (less common).

### Japanese — movement encoding

Japanese does not change particle for direction vs location — に marks both existence (4A) and goal (4B). The verb signals motion.

Goal marker に / へ:
- に for specific goal: 部屋に入る (enter the room)
- へ for general direction: 東京へ行く (go toward Tokyo) — interchangeable with に for most goals

Source marker から: 家から出る (exit the house), 駅から来る (come from the station)

#### Japanese: を for path (critical feature)

Japanese uses を to mark the space traversed — not the direct object of the verb but the space through, across, or along which movement occurs. This is structurally identical to the object marker を but semantically distinct.

- 橋を渡る — cross the bridge (を marks the bridge as the traversed space)
- 公園を歩く — walk through/across the park
- 道を走る — run along the road
- トンネルを通る — go through the tunnel

This surprises learners who expect を to mark only direct objects. Files where this appears should make it visible.

Motion verbs for 4B:

| English | Japanese | Notes |
|---|---|---|
| run | 走る | |
| walk | 歩く | |
| jump | 跳ぶ / 飛び越える | |
| fall | 落ちる | |
| climb | 登る / 上る | |
| enter | 入る | combines with に |
| exit | 出る | combines with から or を |
| cross | 渡る | always を for the crossed space |
| pass through | 通る / 通り抜ける | always を for the traversed space |
| drive | 運転する / 走る | |
| carry | 運ぶ / 持って行く | |
| put | 置く / 入れる | for caused placement |

### Mandarin — directional complements (趋向补语)

Mandarin encodes direction by attaching a directional complement to the motion verb.
This is a compound verb, not a preposition — the direction is lexically built into the verb.

Simple directional complements:
- 进 (enter): 跑进 (run-into), 走进 (walk-into)
- 出 (exit): 跑出 (run-out), 走出 (walk-out)
- 上 (up/onto): 爬上 (climb-onto), 跳上 (jump-onto)
- 下 (down/off): 跳下 (jump-off), 走下 (walk-down)
- 来 (toward speaker): 走过来 (walk-over-here)
- 去 (away from speaker): 走过去 (walk-over-there)

Complex complements: V + 进来/进去/出来/出去/上来/上去/下来/下去
- 跑进去了 (ran into, away from speaker)
- 走出来了 (walked out, toward speaker)

Path prepositions:
- 穿过 (pass through): 穿过公园 (through the park)
- 沿着 (along): 沿着河边走 (walk along the river)
- 绕过 / 绕着 (around): 绕过树 (around the tree)

Source:
- 从 + place + 来/出发: 从家里出来 (came out of the house)
- 离开 (leave): 离开了学校 (left the school)

### 4B contrast pair files

Contrast pair files (frame_id starts with `4b_contrast_`) contain BOTH the static (4A/dative)
and the directional (4B/accusative) forms of the same preposition in the same file,
alternating to make the case flip visible.

Format for contrast pairs: alternate static and directional groups.

```
The cat is under the table.
Die Katze liegt unter dem Tisch.
猫は机の下にいる。
猫在桌子下面。

The cat crawls under the table.
Die Katze kriecht unter den Tisch.
猫は机の下に潜り込んだ。
猫爬到桌子下面。

The book lies under the mat.
Das Buch liegt unter der Matte.
本はマットの下にある。
书在垫子下面。

She slides the book under the mat.
Sie schiebt das Buch unter die Matte.
本をマットの下に滑り込ませた。
她把书滑到了垫子下面。
```

The German case change (dem → den, der → die, dem → das) is the focus.
Each contrast pair file should have 4–6 groups, alternating static and directional.

### 4B motion verb register

Use sein (not haben) for intransitive motion verbs in German Perfekt:
- ist gelaufen, ist gesprungen, ist gefallen, ist geklettert, ist gefahren, ist geschwommen
- ist gegangen, ist gekommen, ist eingetreten, ist geflogen, ist gestiegen

Use haben for caused motion (transitive placement):
- hat gelegt (lay something), hat gestellt (stood something), hat gehängt (hung something),
  hat geworfen (threw), hat geschoben (pushed)

## 4C — Instrument and Means

### What this teaches

The agent uses a tool, body part, or means of transport to accomplish an action.
The cross-linguistic contrasts here are structural:
- English "with" is ambiguous: instrument ("with scissors") = comitative ("with a friend").
- German "mit" is equally ambiguous.
- Japanese で is exclusively instrument/means — comitative uses と.
- Mandarin 用 is a coverb meaning "use/employ" — not used for vehicles.

### Encoding table

| Language | Instrument | Comitative | Vehicle |
|---|---|---|---|
| English | with scissors | with Kate | by train |
| German | mit der Schere | mit Kate | mit dem Zug |
| Japanese | はさみで | ケイトと | 電車で |
| Mandarin | 用剪刀 | 和凯特 | 坐火车 / 骑自行车 |

Mandarin vehicles do not use 用. Instead they use a specific verb:
- 坐 (sit-in): 坐火车, 坐汽车, 坐飞机 (train, car, plane)
- 骑 (ride/straddle): 骑自行车, 骑摩托车 (bicycle, motorbike)
- 开 (drive/operate): 开车, 开船 (car, boat) — when the agent is the operator

### Japanese で vs と distinction

| Particle | Use | Example |
|---|---|---|
| で | instrument / means / material | はさみで切る (cut with scissors) |
| で | location of activity (4A overlap) | 図書館で読む (read at the library) |
| と | comitative (with a person) | 友達と歩く (walk with a friend) |
| と | quotative / enumeration | 「はい」と言う |

This distinction is invisible in English and German but obligatory in Japanese.
Files that show both explicitly (4c_contrast type) should surface this clearly.

### German mit + dative for instrument

mit always takes dative:
- mit der Schere (fem dative)
- mit einem Messer (masc dative)
- mit den Händen (pl dative)
- mit dem Zug (masc dative)
- mit dem Fahrrad (neut dative)

Body-part idioms in German that differ from English:
- mit der Hand schreiben (write by hand)
- zu Fuß gehen (go on foot) — NOT mit dem Fuß
- mit den Augen sehen (see with the eyes) — idiomatic

### Mandarin 用 structure

用 precedes the instrument and the main verb:
- 用剪刀剪纸 (use-scissors cut-paper)
- 用笔写字 (use-pen write-character)
- 用手推 (use-hand push)

用 does NOT appear with vehicles. Use the vehicle-specific verb instead:
- 坐火车去上海 (sit-train go-Shanghai = travel to Shanghai by train)
- 骑自行车上班 (ride-bicycle work = commute to work by bicycle)
- 步行 / 走路 (walk on foot) — no instrument verb needed

### 4C file format

3–5 groups per file. Vary subject, object, and tense across groups.

```
The boy cuts paper with scissors.
Der Junge schneidet Papier mit der Schere.
男の子がはさみで紙を切る。
男孩用剪刀剪纸。

Kate cut the cloth with a knife.
Kate hat das Tuch mit einem Messer geschnitten.
ケイトはナイフで布を切った。
凯特用刀割开了布料。

She will cut the thread with scissors.
Sie wird den Faden mit der Schere schneiden.
糸をはさみで切るだろう。
她会用剪刀剪断线。
```

## Cross-linguistic summary

| Feature | English | German | Japanese | Mandarin |
|---|---|---|---|---|
| Static location | prep + NP | prep + dative NP | NP の localizer に + いる/ある | 在 + NP + localizer |
| Directed movement | prep + NP | prep + **accusative** NP | NP に/へ + motion verb | motion verb + directional complement |
| Path traversal | through/across/along + NP | durch/über/entlang + NP | NP **を** + motion verb | 穿过/沿着/绕过 + NP |
| Source | from/out of + NP | von/aus + dative NP | NP **から** + motion verb | 从 NP + verb / 离开 NP |
| Instrument | with + NP | mit + dative NP | NP **で** + verb | **用** + NP + verb |
| Vehicle (means) | by + vehicle | mit + dative vehicle | vehicle **で** + verb | 坐/骑/开 + vehicle |
| Comitative (with person) | with + person | mit + dative person | person **と** + verb | 和 + person + verb |

## Generation rules

1. Every file: 3–5 groups (4–6 for contrast pairs). Each group: exactly 4 lines (EN/DE/JP/ZH).
2. Blank line between groups. No headers, labels, or grammar notes inside files.
3. German Perfekt for past (hat/ist + participle). Present simple for habitual. werden for future.
4. Japanese plain form. いる/ある distinction is obligatory and must be correct.
5. Japanese を for path is obligatory in path traversal files.
6. Mandarin directional complements (进/出/上/下) for movement into/out of/onto/off.
7. Mandarin vehicle files use 坐/骑/开, never 用.
8. 4B contrast pair files must alternate static (dative) and directional (accusative) groups.
9. Vary subject, verb tense, and reference nouns across groups within a file.
10. Use allowlist words for content nouns.
11. Cross-linguistic differences are features — surface them, do not flatten them.
