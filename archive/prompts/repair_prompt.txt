## CONTEXT

These phase files are the most fundamental units of language instruction for Ninereeds, a small AI trained via Hebbian/association-based learning — not gradient descent. There is no error averaging from scale. Every malformed training pair permanently reinforces a wrong association. These files must be structurally exact, semantically coherent, and free of noise.

The corpus teaches vocabulary in a curriculum: phase 1 = basic concrete physical nouns; phase 2 = built environments and complex nouns; phase 3 = action verbs as observable concepts; phase 4 = biological/life-cycle processes; phase 5 = goal-directed sequences and social/emotional vocabulary (Format B); phase 6 = abstract and cultural concepts.

## FORMAT RULES

### Format A — phases 1, 2, 3, 4, and most of phase 6

File structure: exactly 4 [user]/[Ninereeds] block pairs, **35 lines total** (no trailing blank line).

```
[user]question text here
[Ninereeds]This is [X].
[X] is [body 1].
[X] is [body 2].
[X] is [body 3].
[X] is [body 4].
[X] is [body 5].
[X] is [property A] and [property B].
                             <- blank line (except after block 4)
[user]next question
...
```

Notes:
- `[user]` tag and question on the SAME line, no space: `[user]what does X look like?`
- `[Ninereeds]` tag and opener on the SAME line, no space: `[Ninereeds]This is X.`
- Opener: `This is [X].` — capital T, target word capitalized appropriately
- 5 body lines after opener
- 1 summary line: `[X] is [property A] and [property B].`
- One blank line between block pairs; NO trailing blank line after block 4
- 4 blocks × 8 lines + 3 blank separators = 35 lines total

### Format B — phase 5 vocabulary entries and some phase 6 entries

```
[user]what is [X]?
[Ninereeds][X] is here.
[X] is [definition].
[X] is [location or context].
[X] is [quality or use].
[X] means [definition]. (or: [X] can be [context].)

[user]where is [X]?
...
```

Notes:
- Opener: `[X] is here.` — lowercase X (this is the CORRECT opener for Format B)
- Q2 "where is [X]?" is CORRECT for Format B — do NOT change it

## UNIVERSAL QUALITY RULES (Format A)

1. **No negation**: Body lines must not contain "not", "no", "does not", "cannot", "never", "without", "no X". Rewrite to state what IS true. Example: "does not bend" → "holds its shape"
2. **No pronouns**: No "it", "its", "they", "their", "he", "she". Replace with the target word or a specific noun.
3. **Summary uses "and"**: `[X] is [A] and [B].` — not "or", not "with", not "but"
4. **Affirmative body lines**: Describe what the word IS and DOES, not what it is not.
5. **Consistent subject**: Every body line in a block uses the target word as subject.
6. **Capitalization**: First letter of each body line capital. Target word follows standard capitalization for its type.

---

## COMMON SENSE CHECK — NO MECHANICAL REWRITES

Questions and answers must make genuine sense for the specific target word. Do not do mechanical template substitution if the result is nonsensical:

- "where is a dog?" is nonsensical (implies a single specific location)
- "where can you find a dog?" ✓ (asks about contexts where dogs exist)
- "what does a dog give?" ✓ (dogs give companionship, protection, etc.)
- "what does stiff give?" → write what stiffness contributes: "Stiff gives support to things that hold shape." ✓

When the word is an adjective or abstract concept, adapt the template so it is coherent. The point is that every question-answer pair should be a valid, informative teaching exchange. If a template produces a nonsensical question for a specific word, write the closest sensible phrasing that still follows the template spirit.

## PATCH vs REGENERATE

Damage entries marked **PATCH**: fix only the specific issue(s) listed. Keep all correct content intact.

Damage entries marked **REGENERATE** (or files described as severely truncated/broken): rewrite the full file from scratch. Use the target word (derive it from existing content or the filename). Write meaningful, factually grounded body lines appropriate to the phase. Do not invent facts — write what is generally true about the word.

For REGENERATE files: use phase-appropriate vocabulary in the body lines. Phase 1-3: simple concrete language. Phase 4: biological/process language. Phase 5-6: social, abstract, or functional language.

## FORMAT A DETAILED: HOW TO DETERMINE THE TARGET WORD

The target word (X) for each file is usually visible in the existing content as the repeated subject of body lines. If the file is severely broken, derive the word from: (1) the existing opener, (2) the question text, or (3) a known-good body line.
