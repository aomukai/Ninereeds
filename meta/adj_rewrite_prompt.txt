Ninereeds is a small AI trained via Hebbian/association-based learning. These phase files are its most fundamental vocabulary instruction. Every line permanently reinforces an association — quality matters absolutely.

Adjectives and adverbs are being rewritten to a compact example-based format. Some files previously taught an adverb (e.g., "gradually") — rewrite those to teach the root adjective instead ("gradual"). The filename stays the same; only the content changes.

## FORMAT

Each file is exactly 4 lines. No blank lines. No extra content.

  [user]What does [x] mean?
  [Ninereeds][X] describes something.
  [Positive example 1]. [Positive example 2]. [Positive example 3].
  [Negative example 1]. [Negative example 2]. [Negative example 3].

Rules:
- Line 1: [user]What does [x] mean? — target word is LOWERCASE
- Line 2: [Ninereeds][X] describes something. — target word is CAPITALISED, rest is fixed
- Line 3: three POSITIVE examples — things where the word applies
- Line 4: three NEGATIVE examples — things where the word does NOT apply
- Each sentence ends with a period. Sentences separated by a single space.
- No blank lines anywhere in the file.

CRITICAL — sentence structure:
Every sentence must follow this exact pattern and nothing else:
  [Subject] is [adjective].          ← positive
  [Subject] is not [adjective].      ← negative

No extra clauses. No second predicate. No degree words.
  CORRECT: A library is quiet.
  WRONG:   A quiet library is calm.
  WRONG:   A library is very quiet.
  WRONG:   A library is quiet and peaceful.

Subjects must be short familiar nouns or gerunds. Do not repeat the same subject across line 3 and line 4.

---

## Q4 VARIANT GUIDANCE (for choosing good examples)

Choose examples that make the boundary of the concept clear. For abstract adjectives, use
situations rather than physical objects.

  Perceptual/physical (soft, deep, rough, tight):
    Positive: A pillow is soft. Sand feels rough. A well is deep.
    Negative: A rock is not soft. A smooth road is not rough. A puddle is not deep.

  Behavioral/evaluative (honest, careful, ethical, deliberate):
    Positive: Telling the truth is honest. Checking twice is careful.
    Negative: Hiding facts is not honest. Rushing without thought is not careful.

  Situational/relational (usual, current, proper, separate):
    Positive: Rain in spring is usual. A current plan is in use now.
    Negative: Snow in summer is not usual. A finished plan is not current.

  Abstract/philosophical (real, necessary, complete, exact):
    Positive: A stone is real. Water is necessary for life.
    Negative: A dream is not real in the same way. A guess is not exact.
