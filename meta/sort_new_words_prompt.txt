You are an executor for the Ninereeds training corpus. Perform exactly the following operations and nothing else.

---

## CONTEXT

Ninereeds is a small AI trained via Hebbian association. Its curriculum is organised into 6 phases. Each phase has a distinct complexity level and, for nouns, a distinct file format (see `meta/block_templates.md`). You are sorting new vocabulary words into the correct phase × POS bucket so backfill files can be written.

---

## PHASE GUIDE

### Nouns

| Phase | Word type | Examples |
|-------|-----------|---------|
| 1 | Simple physical objects, natural phenomena | sun, stone, river, dog, leaf, fire, wind |
| 2 | Everyday objects, places, structures | lane, shelf, door, rope, jar, barn, bridge |
| 3 | Abstract concepts, states, processes | destruction, balance, crack, growth, pattern |
| 4 | Materials or phenomena described through an event sequence | wood, frost, flood, ash, ice, smoke |
| 5 | Animals and animate beings | bird, fish, cat, ant, sheep, beetle |
| 6 | Technical, complex, or academic concepts | output, resistance, memory, structure, logic |

### Verbs

| Phase | Word type | Examples |
|-------|-----------|---------|
| 1 | Basic physical actions | run, eat, sleep, jump, push |
| 2 | Everyday transitive actions | carry, clean, open, choose, mix |
| 3 | Process / activity verbs | organise, protect, observe, measure, arrange |
| 4 | Change-of-state verbs | burn, freeze, melt, break, dissolve |
| 5 | Narrative / animate actions | hunt, climb, feed, chase, nest |
| 6 | Abstract, cognitive, or technical | impose, indicate, analyse, represent, resist |

### Adjectives

| Phase | Word type | Examples |
|-------|-----------|---------|
| 1 | Basic sensory / physical | hot, cold, big, small, soft, round, bright |
| 2 | Descriptive / evaluative | narrow, hollow, rough, busy, familiar |
| 3 | Mid-range descriptive | gradual, frequent, partial, dense, steady |
| 4 | Process-related or resultant state | frozen, active, dissolved, cracked |
| 5 | Behavioural / animate | hungry, sleepy, brave, curious, restless |
| 6 | Abstract / complex / academic | implicit, precise, subtle, abstract, logical |

## YOUR TASK

**BATCH SIZE: Process at most 100 words per run. Stop after 100 and report your receipt.**

There are three input files. Work through them in order: nouns first, then verbs, then adjectives.

1. Read `tmp/sort_progress.txt` (if it exists) — this tells you how many words have already been processed per file. Skip those.
2. Read the next 100 unprocessed words from the current input file:
   - `tmp/untaught_nouns.txt`
   - `tmp/untaught_verbs.txt`
   - `tmp/untaught_adjectives.txt`
3. For each word, assign a phase (1–6) using the phase guide above.
4. Append each word to the correct output file:
   - `vocab/phase_N_nouns_new.txt`
   - `vocab/phase_N_verbs_new.txt`
   - `vocab/phase_N_adjectives_new.txt`
   (Create the file if it does not exist. Append if it does.)
5. After finishing the batch, update `tmp/sort_progress.txt`:
   - Format: one line per input file processed so far:
     `nouns: N` / `verbs: N` / `adjectives: N`
   - N = total words processed from that file across all runs
6. Stop after 100 words.

---

## ASSIGNMENT RULES

- Assign the phase that best fits the word's primary meaning and complexity.
- When a word is borderline between two phases, prefer the lower phase (simpler is safer — the model trains bottom-up).
- Multi-word entries (e.g. "fire truck", "bottle of oil"): treat as phase 2 or 3 nouns unless clearly technical (phase 6).
- Numbers as words (e.g. "2", "180"): phase 6 noun.
- Gerund-form words (e.g. "running", "observing") appearing in the noun list: assign as phase 3 noun.
- Do not skip words. Every word in the batch must land in exactly one output file.
