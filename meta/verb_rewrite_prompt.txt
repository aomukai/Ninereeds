## GERUND FORMATION RULES

Convert the base verb to its gerund (-ing) form using standard English spelling:
- Most verbs: add -ing directly (speak → speaking, resist → resisting)
- Verbs ending in silent -e: drop the e, add -ing (impose → imposing, explore → exploring, organize → organizing, decide → deciding, recognize → recognizing)
- Short vowel + single consonant at end: double the final consonant (run → running, sit → sitting, begin → beginning, admit → admitting)
- Verbs ending in -ie: change to -y + ing (die → dying, lie → lying)
- Already a gerund (e.g., "implying"): use as-is

If a word in verbs.txt turns out to be primarily an adjective or noun in context (e.g., "thin", "warm", "near") and does NOT work well as a verb concept, apply the gerund form anyway — "thinning", "warming", "nearing" are all valid action concepts.

## NEW FILE FORMAT

Each file: 4 [user]/[Ninereeds] block pairs, **31 lines total** (no trailing blank line).

IMPORTANT: The gerund form is always lowercase in both questions and answers.
Correct:   `[user]what is exploring?`
Incorrect: `[user]what is Exploring?`

```
[user]what is [xing]?
[Ninereeds][Xing] is [definition sentence].
[Xing] is [description 2].
[Xing] is [description 3].
[Xing] is [description 4].
[Xing] is [description 5].
[Xing] is [property A] and [property B].

[user]when does [xing] happen?
[Ninereeds][Xing] happens when [condition 1].
[Xing] happens when [condition 2].
[Xing] happens [context 3].
[Xing] happens [context 4].
[Xing] happens [context 5].
[Xing] happens [condition A] and [condition B].

[user]what does [xing] bring?
[Ninereeds][Xing] brings [outcome 1].
[Xing] brings [outcome 2].
[Xing] brings [outcome 3].
[Xing] brings [outcome 4].
[Xing] brings [outcome 5].
[Xing] brings [outcome A] and [outcome B].

[user]what is [xing] for?
[Ninereeds][Xing] is for [purpose 1].
[Xing] is for [purpose 2].
[Xing] is for [purpose 3].
[Xing] is for [purpose 4].
[Xing] is for [purpose 5].
[Xing] is for [purpose A] and [purpose B].
```

Rules:
- `[user]` tag and question on the SAME line, no space
- `[Ninereeds]` tag and first body line on the SAME line, no space
- Each [Ninereeds] block: 5 body lines + 1 summary line = 6 lines total after the tag
- Summary line: `[Xing] is [A] and [B].` — must use "and", not "or"
- One blank line between block pairs; NO trailing blank line after block 4
- 4 × (1 user line + 6 ninereeds lines) + 3 blank separators = 31 lines total
- No negation in any line ("not", "no", "does not", "cannot", "never")
- No pronouns ("it", "its", "they", "their", "he", "she")
- All body lines start with capital letter, end with period
- Use vocabulary appropriate to the phase: phase 3 = simple concrete language; phase 6 = slightly more abstract but still clear

---

## CONTENT QUALITY

Write body lines that genuinely describe the concept. Do not pad with vague filler.

For Q1 (what is Xing?): describe the nature and character of the action itself
For Q2 (when does Xing happen?): describe real conditions, contexts, or settings where this action occurs
For Q3 (what does Xing bring?): describe outcomes, results, and effects that follow from the action
For Q4 (what is Xing for?): describe purpose and goals

Use vocabulary appropriate to the phase. Phase 3 words (explore, speak, decide) should use simple language. Phase 6 words (impose, idealize, cooperate) may use slightly more sophisticated phrasing, but keep it clear and concrete.

The summary line for each block should combine two distinct properties from that block's content, not just repeat the same idea twice.
