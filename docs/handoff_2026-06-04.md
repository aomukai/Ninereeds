## Key architecture note (read before touching curriculum)

Ninereeds is a **byte-level Hebbian model** (vocab_size=256, shared weights across
6 layers, xy_sparse = x_sparse * y_sparse co-firing signal). This means:

- File adjacency IS a learning signal — semantically related files adjacent =
  reinforced co-firing clusters. Alphabetical = wrong associations.
- Format matters at the byte level — `[user]` is literally those 6 bytes.
  Training question surface form must match probe surface form exactly.
- No error averaging — at 25M with shared weights, every malformed file or
  wrong ordering has outsized impact. This is why every detail matters.
