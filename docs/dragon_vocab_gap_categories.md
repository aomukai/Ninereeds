# Dragon Vocabulary Gap List by Category

This is a **working target list** of words the dragon should probably learn beyond the current Phase 1–5 curriculum.

It is **not** a dump of every word absent from the wiki. It is a curated list meant to help plan the next bridging batches.

Grounding used:
- `training_data/phases/concept_index.md`
- `training_data/phases/training_sequence.txt`
- `training_data/phases/missing_curriculum_terms.md`
- comparison of Phase 1–5 vocabulary against `training_data/wiki/*_entries.md`

Current Phase 1–5 curriculum facts used for this list:
- 353 training files
- about 1,769 distinct lowercase word forms in the current curriculum corpus
- basic function words like `what`, `is`, `where`, and `does` are present
- important scaffold words like `word`, `sentence`, `thought`, `idea`, `true`, `real`, and `money` are currently missing from Phase 1–5

---

## Priority bands

- **Band A = highest-value bridging vocabulary**
  - needed for wiki comprehension and later reasoning
- **Band B = practical next-layer vocabulary**
  - useful for social, school, and planning behavior
- **Band C = nice to add once the bridge exists**
  - expands expressive range after the core scaffold is in place

## Suggested use

A good next move would be to turn this into three curriculum batches:

1. **Bridge A: language + thought + truth**
   - word, sentence, meaning, thought, idea, true, real, fact, opinion
2. **Bridge B: communication + school + planning**
   - question, answer, say, ask, explain, school, teacher, plan, step, goal
3. **Bridge C: fairness + safety + money + structured comparison**
   - rule, fair, share, safe, danger, money, buy, count, same, different

That would keep the next expansion targeted instead of trying to jump straight toward a generic child-sized vocabulary.

## 13. Minimal pattern grid for the Phase 6 bridge

This is the highest-leverage structural addition for the bridge batch.

Instead of only choosing words, define a **small reusable pattern grid** for each major concept group. That way the dragon does not just see isolated vocabulary items, but also sees the core sentence patterns those words live inside.

### Meta-language

Required pattern examples:
- "A word is a unit of language."
- "A sentence is a group of words."
- "What is a sentence?"
- "This sentence has five words."

### Thought / knowledge

Required pattern examples:
- "A child thinks about the problem."
- "The child knows the answer."
- "The child does not know the answer."
- "How does the child learn?"

### Truth / reasoning

Required pattern examples:
- "This statement is true."
- "This statement is not true."
- "The reason is that ..."
- "Why does this happen? Because ..."

### Communication

Required pattern examples:
- "The child asks a question."
- "The teacher answers the question."
- "The child explains the answer."
- "The student repeats the sentence."

### Planning / sequence

Required pattern examples:
- "First, the child reads."
- "Next, the child writes."
- "The goal is to finish the task."
- "The child follows the steps."

### How to use this grid

For each bridge concept family:
- create 2–4 canonical sentence patterns
- reuse them across multiple files with only small content substitutions
- make sure each target word appears in both question form and statement form when possible
- prefer short, stable frames over clever variation
- use the pattern grid to teach both vocabulary and compositional behavior at the same time

This pattern-grid approach is likely more important than maximizing raw word count, because it teaches the dragon how the new abstract words behave inside ordinary explanatory language.

## 14. Phase 6 bridge vs wiki-entry status
This section adds a practical recommendation for each listed word:
- **Phase status** = whether the term should be taught in a post-Phase-5 bridge curriculum, is already in Phase 1–5, or can wait
- **Wiki status** = whether a dedicated or near-dedicated wiki entry already exists, or whether a new wiki entry should be written

Interpretation rule used here:
- if a word is foundational for understanding later wiki/story/reasoning language, it usually belongs in **Phase 6 bridge** even if a wiki entry already exists
- if a word is narrower or domain-specific, a **wiki entry may be enough**
- some words are already present in Phase 1–5 as raw tokens; that does **not always** mean the concept is fully scaffolded, but it usually means no separate bridge file is urgently needed
