Read the file tmp/rename_list.txt using the read tool. Each line has two tab-separated columns:
  training_data/phases/phase_N/filename.md    [user]question

For each line, identify the single word being asked about — the subject of the question.
Examples:
  "What does the sun look like?" → sun
  "What does running mean?" → running
  "What does smooth mean?" → smooth
  "What happens to a lizard?" → lizard
  "Where can you find a spider?" → spider
  "What do trousers look like?" → trousers
  "What is a diamond?" → diamond

Write the file tmp/rename_map.txt using the write tool. One line per input line, format:
  original_path → word.md

Rules:
- word is lowercase, exactly as it appears in the question (no stemming, no changes)
- If two files in the same phase folder map to the same word, append _2 to the second occurrence, _3 to the third, etc.
- Only the filename changes — do not modify the folder path

Write the complete rename_map.txt now. Do not stop until every line from rename_list.txt has a corresponding entry.
