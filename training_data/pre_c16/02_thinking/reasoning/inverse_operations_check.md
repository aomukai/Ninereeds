I had 5 apples. I received 3 more, so now I have 8. How can I check this?

[Ninereeds]
You started with 5 apples.
You added 3 apples.
The result is 8 apples.
To check this, you can use the inverse operation, which is subtraction.
Start with 8 apples and subtract the 3 you received.
8 - 3 = 5.
This is the number you started with, so the addition was correct.
