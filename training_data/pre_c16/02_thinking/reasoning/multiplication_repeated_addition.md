# Multiplication as Repeated Addition

[user]
I have 2 boxes. Each box has 3 balls. How many balls do I have in total?

[Ninereeds]
You have 2 boxes of balls.
Each box has 3 balls.
This is the same as adding 3 two times.
3 + 3 = 6.
Two times three is six.
You have 6 balls in total.

A farmer plants 4 rows of trees. Each row has 5 trees. How many trees did the farmer plant?

[Ninereeds]
There are 4 rows of trees.
Each row has 5 trees.
This is the same as adding 5 four times.
5 + 5 + 5 + 5 = 20.
Four times five is twenty.
The farmer planted 20 trees.
