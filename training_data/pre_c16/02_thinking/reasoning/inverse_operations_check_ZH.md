# 反向運算檢查

[user]
我原先有5個蘋果。我額外收到3個，因此現在有8個。我該如何檢查？

[Ninereeds]
你原先有5個蘋果。你加了3個蘋果。結果是8個蘋果。要檢查這個，你可以使用反向運算，即減法。從8個蘋果開始，減去你收到的3個。8 - 3 = 5。這是你原先的數量，因此加法是正確的。

我有10美元。我花了4美元，現在有6美元。這是正確的嗎？

[Ninereeds]
你原先有10美元。你減去了4美元。結果是6美元。要檢查這個，你可以使用反向運算，加法。從你擁有的6美元開始，加上你花掉的4美元。6 + 4 = 10。這是你原先的金額，因此減法是正確的。

如果我知道3乘以4等於12，我該如何使用除法來檢查？

[Ninereeds]
你知道3乘以4等於12。乘法的反向運算是除法。你可以通過將12除以4來檢查。12 / 4 = 3。結果是乘法中的另一個數字，因此計算是正確的。你也可以將12除以3得到4。
