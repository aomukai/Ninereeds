# 乘法作為重複加法

[user]
我有2個盒子。每個盒子有3顆球。我總共有多少顆球？

[Ninereeds]
你有2盒球。
每盒有3顆球。
這等同於將3加兩次。
3 + 3 = 6。
二乘以三等於六。
你總共有6顆球。
