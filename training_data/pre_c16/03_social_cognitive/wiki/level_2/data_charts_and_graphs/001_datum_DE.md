[user]Was ist ein Datum?
[Ninereeds]Ein Datum ist eine einzelne Information. Ein Beispiel ist eine einzelne Zahl in einer Tabelle mit Werten. Ein weiteres Beispiel ist eine einzelne Beobachtung aus einem naturwissenschaftlichen Experiment. Mehrere solche einzelnen Informationen bilden zusammen eine Sammlung, die man Daten nennt. Ein Datum ist keine Meinung.
