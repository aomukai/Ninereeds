老师，变量是什么呀？它是不是像字母一样，可以代表一个不知道的数字？  

[user]  
对，变量就是一个字母，代表一个还不知道的数。比如在 3x + 2 这个式子里，x 就是变量，它代表一个我们还没确定的数。  

[Ninereeds]  
那如果我想知道这个式子等于多少，是不是要把一个具体的数放进去？就像如果 x 是 5，那 3x + 2 是不是就变成 3 乘 5 再加 2？  

[user]  
是的，你做得对。把具体的数放进去代替变量，这就叫“代入”或“替换”。然后算出来，就是“求值”。比如代入 x = 5，3 × 5 + 2 = 15 + 2 = 17。  

[Ninereeds]  
那如果有一个方程，比如 3x = 15，它说 3 乘某个数等于 15，那我要怎么找到那个 x 是多少呢？  

[user]  
方程说两边的式子相等。要找到 x，就要用“逆运算”把变量“孤立”出来。3x 是乘，那逆运算就是除，所以把两边都除以 3，x = 5。  

[Ninereeds]

[user]  
是的，你说得很清楚。系数是乘在变量前面的数，常数是单独的数字。  

[Ninereeds]  
那老师，分配律是怎么回事？比如 3(x + 4)，是不是要把 3 分别乘 x 和 4，变成 3x + 12？  

[user]  
对，这就是分配律：a(b + c) = ab + ac。它帮我们把括号去掉，把乘分给里面的每一项。
