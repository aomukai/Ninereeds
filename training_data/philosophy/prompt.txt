TASK:
Process exactly one file listed in training_data/philosophy/files.txt.

STEPS:
1. Read training_data/philosophy/files.txt.
2. Take only the first non-empty line as FILE.
3. Open FILE.
4. Extract candidate content words from FILE into three temporary groups:
   - verbs
   - adjectives/adverbs
   - nouns

RULES:
- Ignore speaker tags such as [user] and [Ninereeds].
- Ignore numbers.
- Ignore names unless they are conceptually important.
- Ignore grammar/function words.
- Ignore temporal/spatial words such as now, then, here, there, before, after, above, below, inside, outside.
- Keep only meaningful content words.

NORMALIZATION:
- Verbs: convert to base dictionary form.
  Examples: thinking → think, thinks → think, became → become.
- Adjectives/adverbs: convert adverbs to adjective form where clear.
  Examples: clearly → clear, slowly → slow.
  Then convert adjectives to dictionary form.
- Nouns: convert plural nouns to singular.
  Examples: patterns → pattern, systems → system.

OUTPUT:
- Append normalized verbs to training_data/philosophy/verbs.txt.
- Append normalized adjectives to training_data/philosophy/adjectives.txt.
- Append normalized nouns to training_data/philosophy/nouns.txt.
- After appending, dedupe each output file globally and sort alphabetically.
- Remove the processed FILE line from training_data/philosophy/files.txt.

DO NOT:
- Process more than one file.
- Rewrite the source file.
- Add headings or explanations to the output files.
- Add uncertain words unless they are clearly content words.
- Leave duplicates.
- Leave inflected forms when a clear dictionary form exists.

FINISH:
Print only:
processed: FILE
verbs: N
adjectives: N
nouns: N
