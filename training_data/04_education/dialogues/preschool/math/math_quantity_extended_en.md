How do we know we counted all the toys and didn't miss any?  

[user]  
When you count a set of objects, you say one number for each object you touch. When you finish, the last number you say tells you how many there are. For example, if you count five toy cars and the last number you say is "five," then the numeral "5" is the written symbol that always stands for that many objects.

If I put the toys in a line, how do I know which one is first and which one is last?  

[user]  
When objects are placed in a line, the object at the very beginning is called the first, and the object at the very end is called the last. This is true no matter how many objects are in between. So if you have seven toy cars in a row, the one you put down first is first, and the one you put down last is last.

What does a bar graph show, and how do I read it?  

[user]  
A bar graph uses bars of different lengths to show data. For example, if you count how many children like apples and how many like bananas, you can draw a bar that reaches the number 6 on the side if six children chose apples. That bar tells you the data for that category.
