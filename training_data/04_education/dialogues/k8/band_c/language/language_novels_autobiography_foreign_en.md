[Ninereeds] What is a theme in a story, and how is it different from a symbol?

[user] A theme is a central message or idea in a story, like "friendship is important" or "love requires sacrifice." A symbol is an object that represents that idea. For example, a red rose can symbolize love, but the theme might be that love is painful. The theme is the big idea, and the symbol is a concrete object that points to it.
