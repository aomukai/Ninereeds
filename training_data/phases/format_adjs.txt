# Phase file format — ADJECTIVE entries
# Created 2026-05-06

---

## STRUCTURE

4 lines. No blank lines.

  [user]What does [x] mean?
  [Ninereeds][X] describes something.
  [Pos 1]. [Pos 2]. [Pos 3].
  [Neg 1]. [Neg 2]. [Neg 3].

---

## SENTENCE RULE

Every sentence on line 3 and line 4 must follow this exact pattern:

  [Subject] is [adjective].          ← positive
  [Subject] is not [adjective].      ← negative

Nothing else. No extra clauses. No second predicate.

  CORRECT: A library is quiet.
  WRONG:   A quiet library is calm.      (adjective used as modifier, not predicate)
  WRONG:   A library is quiet and calm.  (extra words)
  WRONG:   A library is very quiet.      (degree word added)

Subjects should be short and familiar — single nouns or simple noun phrases.
Gerunds are allowed as subjects: Stealing is not ethical. Helping is ethical.

---

## EXAMPLES

  [user]What does strong mean?
  [Ninereeds]Strong describes something.
  A bridge is strong. A rope is strong. Iron is strong.
  Paper is not strong. A bubble is not strong. Smoke is not strong.

  [user]What does quiet mean?
  [Ninereeds]Quiet describes something.
  A library is quiet. A lake is quiet. A forest is quiet.
  A market is not quiet. A crowd is not quiet. A machine is not quiet.

  [user]What does gradual mean?
  [Ninereeds]Gradual describes something.
  Change is gradual. Growth is gradual. Learning is gradual.
  An explosion is not gradual. A break is not gradual. A fall is not gradual.

  [user]What does ethical mean?
  [Ninereeds]Ethical describes something.
  Honesty is ethical. Helping is ethical. Keeping a promise is ethical.
  Stealing is not ethical. Lying is not ethical. Hurting is not ethical.
