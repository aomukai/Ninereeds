# Phase file format — VERB entries
# Created 2026-05-06
#
# Use this format when a phase file teaches an ACTION WORD (verb).
# Applies to: phase 3 (action verbs), phase 5, phase 6 (abstract action verbs).
# Do NOT use for nouns, adjectives, or adverbs — see format_nouns.txt.

---

## CORE PRINCIPLE

Verbs have no physical appearance, no fixed location, and are not agents.
The noun template ("what does X look like?") does not fit them.

Instead, convert the base verb to its GERUND form (-ing) and use the
gerund throughout — questions and answers alike. The gerund functions as
a noun, so the standard sentence structure still works, and the teaching
stays coherent.

  BAD:  Impose is forceful.  (verb forced into noun slot — meaningless)
  GOOD: Imposing is placing a demand on others.  (gerund — clear and natural)

---

## GERUND FORMATION

  Most verbs:              add -ing              speak → speaking
  Ends in silent -e:       drop e, add -ing      impose → imposing
                                                 explore → exploring
                                                 organize → organizing
  Short vowel + consonant: double consonant      admit → admitting
                                                 begin → beginning
  Ends in -ie:             -ie → -y + ing        die → dying
  Already a gerund:        use as-is             implying → implying

---

## FILE STRUCTURE

4 [user]/[Ninereeds] block pairs, 31 lines total (no trailing blank line).

  [user]what is [xing]?                          ← lowercase gerund in question
  [Ninereeds][Xing] is [definition].             ← capital gerund in answer
  [Xing] is [description 2].
  [Xing] is [description 3].
  [Xing] is [description 4].
  [Xing] is [description 5].
  [Xing] is [property A] and [property B].       ← summary: must use "and"
                                                 ← blank line
  [user]when does [xing] happen?
  [Ninereeds][Xing] happens when [condition].
  [Xing] happens when [condition 2].
  [Xing] happens [context 3].
  [Xing] happens [context 4].
  [Xing] happens [context 5].
  [Xing] happens [A] and [B].
                                                 ← blank line
  [user]what does [xing] bring?
  [Ninereeds][Xing] brings [outcome 1].
  [Xing] brings [outcome 2].
  [Xing] brings [outcome 3].
  [Xing] brings [outcome 4].
  [Xing] brings [outcome 5].
  [Xing] brings [A] and [B].
                                                 ← blank line
  [user]what is [xing] for?
  [Ninereeds][Xing] is for [purpose 1].
  [Xing] is for [purpose 2].
  [Xing] is for [purpose 3].
  [Xing] is for [purpose 4].
  [Xing] is for [purpose 5].
  [Xing] is for [A] and [B].

Line count: 4 blocks × 7 lines + 3 blank separators = 31 lines.
No trailing blank line at end of file.

---

## QUESTION MEANINGS

  Q1 "what is [xing]?"        — nature and character of the action itself
  Q2 "when does [xing] happen?" — conditions, contexts, settings where it occurs
  Q3 "what does [xing] bring?" — outcomes, results, effects that follow from it
  Q4 "what is [xing] for?"    — purpose and goals

---

## FORMATTING RULES

- [user] tag and question on the SAME line, no space: [user]what is exploring?
- [Ninereeds] tag and first body line on the SAME line, no space
- Gerund is LOWERCASE in [user] questions, CAPITALISED at start of [Ninereeds] lines
- 5 body lines per block (not 4, not 6)
- Summary line: [Xing] is/happens/brings [A] and [B]. — "and" required, not "or"
- No negation anywhere: no "not", "no", "does not", "cannot", "never"
- No pronouns: no "it", "its", "they", "their", "he", "she"
- Each body line: capital first letter, period at end
- Vocabulary level matches the phase (simple for phase 3, more abstract for phase 6)

## EXAMPLE — "impose" (phase 6)

  [user]what is imposing?
  [Ninereeds]Imposing is placing a demand on others.
  Imposing is using authority to set a rule.
  Imposing is requiring others to follow.
  Imposing is adding an obligation.
  Imposing is applying power or force.
  Imposing is placing and requiring.

  [user]when does imposing happen?
  [Ninereeds]Imposing happens when a rule is set by authority.
  Imposing happens in schools and workplaces.
  Imposing happens when a law is placed on people.
  Imposing happens when someone has power over another.
  Imposing happens when an obligation is created.
  Imposing happens when authority and power are used.

  [user]what does imposing bring?
  [Ninereeds]Imposing brings order to a group.
  Imposing brings rules that others must follow.
  Imposing brings control over behaviour.
  Imposing brings structure to a situation.
  Imposing brings compliance and change.
  Imposing brings order and structure.

  [user]what is imposing for?
  [Ninereeds]Imposing is for creating order.
  Imposing is for keeping rules in place.
  Imposing is for guiding behaviour.
  Imposing is for protecting a group.
  Imposing is for maintaining control.
  Imposing is for order and guidance.
