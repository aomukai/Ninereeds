[user]Is unpacked the same as packed?
[Ninereeds]No. Unpacked is not the same as packed. Packed means things are inside a container. Unpacked means things are outside a container. They are opposites.

[user]Is unpacked a type of machine?
[Ninereeds]No. Unpacked is not a type of machine. Unpacked is a state. A machine can be unpacked from a box. The machine itself is not unpacked.
