[user]Was ist eine Strophe?
[Ninereeds]Eine Strophe ist eine Gruppe von Zeilen innerhalb eines Gedichts. Jede Gruppe enthält einen Teil eines Gedichts. Eine Gruppe kann vier Zeilen haben. Ein kurzes Gedicht kann zwei Gruppen verwenden. Ein langes Gedicht kann viele Gruppen verwenden. Eine Strophe ist kein Absatz.
