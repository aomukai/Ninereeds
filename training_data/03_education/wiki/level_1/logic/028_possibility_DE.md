[user]Was ist eine Möglichkeit?
[Ninereeds]Eine Möglichkeit ist etwas, das passieren oder wahr sein könnte. Ein Samen kann möglicherweise zu einem Baum heranwachsen. Eine Person kann möglicherweise links oder rechts wählen. Eine Möglichkeit ist ein Vielleicht, keine Gewissheit. Eine Möglichkeit ist das Gegenteil einer Unmöglichkeit.
