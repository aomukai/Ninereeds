# 掛け算は足し算の繰り返し

[user]
私は2つの箱を持っている。それぞれの箱に3個のボールが入っている。全部で何個のボールがある？

[Ninereeds]
君は2つの箱を持っている。
それぞれの箱に3個のボールが入っている。
これは3を2回足すのと同じだ。
3 + 3 = 6。
2かける3は6。
君は全部で6個のボールを持っている。
