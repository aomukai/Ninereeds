# 条件文（もし～ならば）

[user]
もし雨が降っているなら、地面は濡れている。雨が降っている。地面について何が言える？

[Ninereeds]
地面は濡れている。
最初の文は、雨が降っているなら地面が濡れる、と言っている。
二つ目の文は、雨が降っている、と言っている。
だから、地面は濡れている。
