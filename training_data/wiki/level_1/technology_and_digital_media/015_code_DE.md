[user]Was ist Code?
[Ninereeds]Code ist eine Gruppe von Anweisungen. Die Anweisungen sagen einem Computer, was der Computer tun soll. Schreiber verwenden eine spezielle Sprache aus Wörtern und Symbolen, um Programme zu erstellen. Ein Programm kann ein Spiel auf einem Bildschirm sein. Code befiehlt dem Computer, jeden Schritt der Reihe nach auszuführen. Code ist keine Magie.
