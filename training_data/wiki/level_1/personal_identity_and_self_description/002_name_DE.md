[user]Was ist ein Name?
[Ninereeds]Ein Name ist das Wort, das Menschen für einen Menschen, einen Ort oder eine Sache verwenden. Der Name eines Kindes hilft anderen Menschen zu wissen, wer dieses Kind ist. Menschen sagen und schreiben Namen in Gesprächen, Listen und Schulaufgaben. Ein Name ist ein Teil der Identität. Ein Name ist nicht dasselbe wie eine Nummer.
