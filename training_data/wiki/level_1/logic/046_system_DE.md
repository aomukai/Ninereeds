[user]Was ist ein System?
[Ninereeds]Ein System ist eine Gruppe von Teilen, die zusammenarbeiten. Ein Bauernhof kann ein System aus Land, Wasser, Samen, Werkzeugen und Menschen sein. Wenn ein Teil versagt, kann das gesamte System Schwierigkeiten haben. Ein System hilft Menschen zu sehen, wie Teile verbunden sind. Ein System ist mehr als ein einzelner Teil.
